KaBa - Danis kategorisiertes Bilderarchiv.

Eigentlich habe ich es im Internet gefunden (es heisst JCap) und dann für meine Zwecke modifiziert. Ich brauchte eine Software, mit der man auf CD exportierte Bilder weiterhin mit Kategorien absuchen kann. Dadurch, dass dieses Programm in JAVA geschrieben wurde, läuft es mühelos unter allen Betriebs-Systemen (nur die JAVA-VM muss installiert sein, das ist sie aber, wenn du diesen Text lesen kannst :-) ) und kann dadurch, dass es klein ist, gleich auf alle Bilder-CDs mit gebrannt werden. Auch mit Rechten gibt es keine Probleme, es ist frei kopierbar und dank offenliegender Quellen (Open Source) sogar - für den, der es will - modifizierbar. Man sagt, es unterliegt der GPL - GNU Public License. Verkaufen darf man es übrigens nicht mehr, sobald eine Zeile freier Code verwendet wird.

Man kann damit super:

  • Bilder betrachten.
  • Bilder beschriften, z.B. in Description kann man ganze Geschichten dazu schreiben.
  • Bilder kategorisieren: mit Stichworten (Kategorien) versehen.
  • Später kann man dann nach diesen Stichworten die Bilder gezielt suchen und schnell finden.
  • Gesucht wird in allen Verzeichnissen unterhalb der angegebenen Location.
  • Im rechten Panel sieht man interessante Hintergrund-Daten, die aus dem Bild selber gelesen werden (die genaue Aufnahmezeit, Belichtung etc.).
  • Gespeichert werden die Texte (Geschichten, Überschriften und Kategorien) in unterschiedlichen Text-Dateien in den Bildverzeichnisse. Dadurch sind die Text-Daten auch nach Verschieben ganzer Verzeichnisse oder Brennen auf eine CD weiterhin vorhanden!

Wenn du möchtest, dass das Programm auf seine letzten Einstellungen bei Bildschirmdarstellung und -aufteilung und bei der Auswahl der Verzeichnisse zurückgreifen kann, dann erlaube ihm, eine Initialisierungs-Datei auf deiner Festplatte zu schreiben. Diese kannst du später auch in das Arbeitsverzeichnis von KaBa umziehen. Meld dich, wenn dir Fehler auffallen oder du weitere Ideen zum Programm hast!
