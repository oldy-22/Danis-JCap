Hallo zusammen,

das ist eine kurze Info zur Handhabung des Programms KaBa - Danis kategorisiertes 
Bilderarchiv.

Eigentlich habe ich es im Internet gefunden (es heisst JCap) und dann für meine
Zwecke modifiziert. Ich brauchte eine Software, mit der man auf CD exportierte
Bilder weiterhin mit Kategorien absuchen kann. Dadurch, dass dieses Programm 
in JAVA geschrieben wurde, läuft es mühelos unter allen Betriebs-Systemen (nur
die JAVA-VM muss installiert sein, das ist sie aber, wenn du diesen Text lesen
kannst :-) ) und kann dadurch, dass es klein ist, gleich auf alle Bilder-CDs mit
gebrannt werden. Auch mit Rechten gibt es keine Probleme, es ist frei kopierbar 
und dank offenliegender Quellen (Open Source) sogar für den, der es will
modifizierbar. Man sagt, es unterliegt der GPL - GNU Public License.

Man kann damit super:
- Bilder betrachten
- Bilder beschriften, z.B. in Description kann man ganze Geschichten dazu schreiben
- Bilder kategorisieren
- später kann man dann nach Stichworten suchen

Wenn du möchtest, dass das Programm auf seine letzten Einstellungen bei Bildschirmdarstellung
und -aufteilung und bei der Auswahl der Verzeichnisse zurückgreifen kann, dann erlaube ihm,
eine Initialisierungs-Datei auf deiner Festplatte zu schreiben. Diese kannst du auch,
in das beschreibbare Arbeitsverzeichnis von KaBa später umziehen.
